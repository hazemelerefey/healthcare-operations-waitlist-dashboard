Healthcare Operations & Waitlist Dashboard

🏥 Overview
A sophisticated operational dashboard built to track and optimize patient waitlists. This project focuses on the "heavy lifting" of data engineering, utilizing a star schema and automated ETL pipelines to handle large-scale healthcare data with precision.

🏗️ Data Architecture
Data Modeling: Implemented a Star Schema for optimized query performance.

ETL Pipeline: Automated Extraction, Transformation, and Loading (ETL) to ensure data integrity and real-time updates.

Tools: Power BI, SQL for initial data mining, and Power Query.

📈 Analytical Insights
Waitlist Profiling: Analysis of patient age profiles (0-15, 16-64, 65+) against time bands (18+ months vs. 0-3 months).

Case Type Split: Granular breakdown of Inpatient, Day Case, and Outpatient data.

Trend Analysis: Historical monthly tracking comparing current waitlists (709K) against previous years to identify seasonal strain.

Specialty Focus: Ranking of top medical specialties (Dermatology, ENT, Orthopaedics) by waitlist volume.
